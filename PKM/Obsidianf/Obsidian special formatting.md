[[Obsidian]] uses industry-standard Commonmark, GFM, Github [[wikilinks]], and [[YAML front matter]]. The only Obsidian-specific extensions it adds are:

```txt
`%%` for comment blocks
`^blockid` for block identifiers
`#tagname` for tags
`$` for math markup
`==highlighted text==`
```
