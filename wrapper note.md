An [[Obsidian note]] between 2 things.

> [!example]
> instead of directly linking to a website, i link to a note about the website.
> and in that note, i link to the website.
> ```mermaid
> graph LR
>  A[Note] --wikilink--> B[Note about website] --URL--> C[Website]
> ```

A wrapper note lets me write my thoughts about the [[website]], and also track the website as a note in my [[Obsidian vault]], using [[autocomplete]], or [[backlink|backlinks]].
