Recently I started writing more notes that start with the date.

The biggest win is the reduction in [[mental resistance|mental friction]].
- It removes the option to think of a good title. I just write the date and start writing a note, then a theme emerges, and I add it to the note name, similar to [[random title note flow]].
- There's no need to think of a unique title. 2 notes can have the same title, but different dates.

I expected it might makes it easier to see if a note is relevant. Recent notes tend to be more relevant. But I haven't yet encountered this as useful part so far. Maybe the odd time?
