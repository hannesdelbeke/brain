

| Cable Type              | Power Delivery?     | Typical Wattage | Notes                        |
| ----------------------- | ------------------- | --------------- | ---------------------------- |
| **[[USB-C]] (USB‑PD)**  | ✅ Yes               | Up to 240W      | Main modern power standard   |
| **[[USB-A]]**           | ⚠️ Limited          | 2.5–12W         | Depends on charger/port      |
| **[[Thunderbolt]] 3/4** | ⚠️ Yes (via USB‑PD) | Up to 100–240W  | Not native to Thunderbolt    |
| **[[HDMI]]**            | ❌ Practically no    | ~0.25W          | Only for internal signaling  |
| **[[DisplayPort]]**     | ❌ Practically no    | ~1.65W          | Not enough for devices       |
| **[[Ethernet]] (PoE)**  | ✅ Yes (if PoE)      | Up to 90W       | Requires PoE switch/injector |
| **Coax**                | ⚠️ Special cases    | 13–18V          | Only for antennas/satellite  |
| **VGA/DVI**             | ❌ No                | 0W              | No power lines               |
| **Audio cables**        | ❌ No                | 0W              | Signal only                  |

### Emerging Standard: GPMI
A new proposed cable standard called **GPMI** claims:
- Up to **480W** power delivery
- High‑bandwidth video + data
- Aims to replace HDMI, DisplayPort, and Thunderbolt
But it’s not in any consumer products yet.
