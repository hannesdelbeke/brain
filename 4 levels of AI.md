came across a youtube vid that suggested to learn AI in this order, and spend at least a few months in each phase to learn about the limitations of AI.

1. chat in browser - [[chat]]
	   manually copy paste notes and code
	   back-and-forth convo between you and AI
2. suggest code in [[integrated development environment|IDE]] - [[copilot]]
	   you work with AI in the IDE and read the code block suggestions before you accept them
3. AI in [[command line interface|CLI]]
	   you chat with AI, go back and forward. instruct it for each step
4. fully [[autonomous]] ai - [[automate]]
	   You go to sleep, give it instructions, come back day after and it did everything.
